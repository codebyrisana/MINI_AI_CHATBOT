# MINI AI CHATBOT
### Simple AI Chatbot using Python Core Features

##  Project Overview
This project is a mini AI chatbot built using **Python Core Features**. It can read user messages, identify keywords, and provide proper responses.

##  Key Features
* **Keyword-based Responses:** Bot identifies specific keywords to answer.
* **File Logging:** Automatically saves the conversation in `chat_history.txt`.
* **Exit Option:** Use keywords like `bye` or `quit` to end the chat.

## Code Structure
* `chatbot_basic.py`: Basic chatbot with simple responses.
* chatbot_file.py`: Advanced version with file logging and multiple keywords.
* `chat_history.txt`: Automatically saves conversation history.

## Future Scope
* Adding **Machine Learning** and **NLP** for smarter responses.
* **Voice Recognition** (Speech-to-Text & Text-to-Speech).
* Developing a **GUI interface** using Tkinter.
